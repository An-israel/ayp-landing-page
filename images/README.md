# Images

Place the following image files in this directory:

| Filename       | Description                                      |
|----------------|--------------------------------------------------|
| cover.jpg      | Book front cover (used in 3D book mockup)        |
| mockup1.jpg    | Book mockup image 1 (shown in hero grid)         |
| mockup2.jpg    | Book mockup image 2 (shown in hero grid)         |
| author.jpg     | Author photo (used in "About the Author" section)|

Supported formats: .jpg, .jpeg, .png, .webp
